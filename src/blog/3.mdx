## 每个人看待问题的角度都好不一样

对于大部分人来说, 前端要学的东西多而且杂, 但其实也只是表象.其实我并不认为前端要学的东西多,并且我常常感到乏力以及不知道学什么.

说到底前端的语言 html\css\js 根基不变,看起来有很多的框架要学,但实际会发现相同点特别多.

尤其我现在做的这些业务,有一些是关于低代码的, 我就在想 html 本身就是一个语义化的文本,为什么我需要存一份 json 语义化的文本,再转化成语义化的文本呢?

为什么我认为这些 json 不算做数据, 因为他本质就是一个 html ,如果可以的话后端其实并不想存这些只用于显示逻辑的数据.

## 另外我一直也不认同要将前端学精

但也不是说浅显的学习一下, 可以说稍微保守的学习.

> 需要注意的是资源非常匮乏,大部分角色只能拉倒 40 级

这里我拿原神举个例子, 原神中有很多角色, 你想要加强或者提高则需要付出相应的资源, 而你的资源是有限的.

如果没有特别感兴趣的方向,培养大家普遍认可的,或者就业机会很好. 但是并不意味着你不需要培养别的角色,

因为环境在变,你的角色很有可能在某个环境下不适用,或者需要更高练度才能适应.

诶这下是不是就能听明白了, 没想到原神还有这种用处~
